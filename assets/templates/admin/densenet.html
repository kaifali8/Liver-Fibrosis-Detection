DenseNet (Densely Connected Convolutional Network)

DenseNet is a deep learning model that improves information flow between layers by introducing dense connections. Unlike traditional CNNs, where each layer passes information to the next sequentially, DenseNet connects every layer to every other layer, enhancing feature reuse and reducing the number of parameters.
